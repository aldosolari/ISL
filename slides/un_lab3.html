Interaction Terms

  • Include interaction terms with :
  • Example: lstat:age
  • Use * as shorthand for:
    • main effects
    • interaction term
  • lstat * age is shorthand for:
    • lstat + age + lstat:age
summary(lm(medv ~ lstat * age, data = Boston))

Writing Functions

  • R includes many built-in functions

  • We can also write our own functions when needed

  • Example: create a function LoadLibraries()

  • Before defining it, calling the function gives an error

LoadLibraries
Error: object 'LoadLibraries' not found
LoadLibraries()
Error in LoadLibraries(): could not find function "LoadLibraries"
  • Define the function with function() { ... }
  • The braces {} contain the commands in the function body
LoadLibraries <- function() {
 library(ISLR2)
 library(MASS)
 print("The libraries have been loaded.")
}
  • Typing the function name prints its definition
LoadLibraries
<srcref: file "" chars 1:18 to 5:1>
  • Calling the function executes the commands
LoadLibraries()

Choosing Among Models Using the Validation-Set Approach and Cross-Validation

  • Model size can also be selected using:

    • validation set
    • cross-validation

  • Important: all model-fitting steps must use only the training data

  • Variable selection must also be performed within the training set

  • Create a training/test split

  • train is TRUE for training observations

  • test is TRUE for test observations

set.seed(1)
train <- sample(c(TRUE, FALSE), nrow(Hitters),
    replace = TRUE)
test <- (!train)
  • Perform best subset selection on the training data only
regfit.best <- regsubsets(Salary ~ .,
    data = Hitters[train, ], nvmax = 19)
  • Build a model matrix for the test data
test.mat <- model.matrix(Salary ~ ., data = Hitters[test, ])
  • Compute validation error for each model size
  • For each i:
    • extract coefficients
    • compute predictions
    • compute test MSE
val.errors <- rep(NA, 19)
for (i in 1:19) {
 coefi <- coef(regfit.best, id = i)
 pred <- test.mat[, names(coefi)] %*% coefi
 val.errors[i] <- mean((Hitters$Salary[test] - pred)^2)
}

  • regsubsets() has no built-in predict() method
  • Define a custom prediction function for reuse
predict.regsubsets <- function(object, newdata, id, ...) {
  form <- as.formula(object$call[[2]])
  mat <- model.matrix(form, newdata)
  coefi <- coef(object, id = id)
  xvars <- names(coefi)
  mat[, xvars] %*% coefi
}

  • Now use cross-validation for model selection

  • Create:

    • 10 folds
    • a matrix to store CV errors
k <- 10
n <- nrow(Hitters)
set.seed(1)
folds <- sample(rep(1:k, length = n))
cv.errors <- matrix(NA, k, 19,
    dimnames = list(NULL, paste(1:19)))
  • For each fold:
    • fit best subset selection on training folds
    • predict on the validation fold
    • store the MSE for each model size
for (j in 1:k) {
  best.fit <- regsubsets(Salary ~ .,
       data = Hitters[folds != j, ],
       nvmax = 19)
  for (i in 1:19) {
    pred <- predict(best.fit, Hitters[folds == j, ], id = i)
    cv.errors[j, i] <-
         mean((Hitters$Salary[folds == j] - pred)^2)
   }
 }
